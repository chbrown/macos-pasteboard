# macos-pasteboard

Like OS X's built-in `pbpaste` but more flexible and raw.


## Install

```sh
git clone https://github.com/chbrown/macos-pasteboard
cd macos-pasteboard
make install
```


## Example

Highlight and copy the `macos-pasteboard` element in the preceding section.

```sh
pbv
```
